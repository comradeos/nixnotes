
В выводе команды df -h не отображается информация о флешке. Это может означать, что флешка не подключена или не смонтирована. Давайте проверим, как подключить и смонтировать флешку, а затем снова проверим свободное место.

Шаги для проверки свободного места на флешке

lsblk

Эта команда покажет список всех блоковых устройств. Найдите устройство, которое соответствует вашей флешке (например, /dev/sdb или /dev/sdc).

Если флешка не смонтирована автоматически, смонтируйте ее вручную:

sudo mount /dev/sdX1 /mnt

Замените /dev/sdX1 на соответствующее устройство, найденное на предыдущем шаге. /mnt — это точка монтирования, вы можете использовать любую другую подходящую директорию.

Проверьте свободное место на флешке:

df -h /mnt
df -h /media/io/_IO32/

Yocto Project. Yocto Project — это набор инструментов и процессов для создания кастомных дистрибутивов Linux для встроенных систем. Вот что означают указанные требования:

Требования к системе для сборки (Build Host)
Свободное место на диске:

Минимум 90 ГБ свободного места на диске. Больше свободного места поможет запускать несколько сборок и повысить производительность за счет повторного использования артефактов сборки.
Оперативная память (RAM):

Минимум 8 ГБ оперативной памяти. Современный хост для сборки с максимальным количеством оперативной памяти и ядер процессора рекомендуется для максимальной производительности сборки.
Поддерживаемая Linux-дистрибуция:

Система должна работать на поддерживаемой Linux-дистрибуции, такой как последние версии Fedora, openSUSE, CentOS, Debian или Ubuntu. Для получения списка поддерживаемых дистрибутивов Linux, обратитесь к разделу "Supported Linux Distributions" в Yocto Project Reference Manual. Для получения подробной информации о подготовке хоста для сборки, обратитесь к разделу "Preparing the Build Host" в Yocto Project Development Tasks Manual.
Требования к установленным программам
Git:

Версия 1.8.3.1 или выше.
tar:

Версия 1.28 или выше.
Python:

Версия 3.8.0 или выше.
gcc:

Версия 8.0 или выше.
GNU make:

Версия 4.0 или выше.
Пример установки необходимых программ на Ubuntu
Для установки или обновления этих программ на Ubuntu, выполните следующие команды:

sudo apt update
sudo apt install git tar python3 gcc make
